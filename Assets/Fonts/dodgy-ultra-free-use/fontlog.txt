FONTLOG for the Dodgy Ultra font

This file provides detailed information on the Dodgy Ultra Font Software.
This information should be distributed along with the Dodgy Ultra font and any derivative works.

Basic Font Information

Dodgy Ultra is a Unicode typeface that supports most languages that use the Latin script and its variants, and could be expanded to support other scripts.
